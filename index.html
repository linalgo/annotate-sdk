linalgo-annotate documentation¶

linalgo-annotate is a Python library for natural language processing built around annotations. It is compatible with the W3C web annotation data model.

Contact: arnaud@linalgo.com

Installation¶

linalgo-annoate is compatible with python 3.6 and above (see Installation).

Getting task data

from linalgo.client import LinalgoClient

client_id = '<YOUR_CLIENT_ID_HERE>'
client_secret = '<YOUR_SECRET_HERE>'
linalgo_client = LinalgoClient(client_id, client_secret)
