Installation¶

linalgo-annoate is currently compatible with python 3.6 and above.

Using pip¶

Coming soon. linalgo-annotate cannot be installed from PyPi yet.

Manual installation¶

Cloning the repository

linalgo-annotate is available from:

https://github.com/linalgo/linalgo-annotate.git

Dependencies

You can install the required dependencies with the following command:

pip install -r requirements.txt

Development mode

Currently, the library must be installed in develpment mode to function properly:

pip install -e .

Testing the installation

Verify that the installation is by running the folowing command at the root of the project:

pytest
